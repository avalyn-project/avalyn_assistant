What is Avalyn?
Avalyn is a Discord assistant that can do more than simply help you with the
moderation, organization, and overall management of you Discord servers and channels.
Avalyn can also
  • Parse SHA256 Data.
  • Parse Base58 Data.
Avalyn can also assist you with creating Genesis Blocks for your own Blockchains and Crytpocurrencies
via the Prototype Alice Genesis Block Creation Plugin created by The Avalyn Project.

There are so many Avalyn commands and features to come in future updates.

[Avalyn Server Commands]:

[Utility Commands]:
  • /user - This command simply displays the information of command user. (Private use only)
  • /ping - This command simply displays the current Latency/Ping of Avalyn.
  • /server - This command simply displays the current server's information.
  • /specs - This command displays information and specifications regarding Avalyn.
  • /purge - This command simply removes (purges) a specific amount of messages from a specific Discord channel.


[Moderation Commands]:
  • /kick - This command simply kicks a specific user from a Discord server.
  • /ban - This command simply bans a specific user from a Discord server.
  • /timeout - This command simply puts a specific user in a timeout.


[Blockchain Commands]:
  • /genesis - This command allows you to generate a Genesis Block for your own Blockchain. (Private use only)
  • /print - This command simply displays the block hashes of specified Avalyn Blockchain blocks.
  • /height - This command simply displays the current height of the Avalyn Blockchain.
